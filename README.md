## Quick Guide

This app helps you calculate a compact Heat Island layout automatically.

### 1. Choose a template or create your own island

Open **Templates** to select one of the available island maps.

At the moment, there are only **4 template maps** available. They are marked with numbers from **1 to 4** at the beginning of the template list.

Other maps are not included yet because I simply have not unlocked those islands in the game.

If you want to create your own island manually, open **Drawing** mode. There you can fully draw and edit the island by hand.

On the map:

* Green cells = island
* Blue cells = water / unavailable area
* Brown cells = bushes / rocks
* Orange cells = source
* Yellow cells = generator
* White cells = heat sink

### 2. Set building types and levels

Open **Buildings** and select:

* Source type and level
* Generator type and level
* Heat sink type and level

These values are used for the calculation.

### 3. Run auto-placement

Go back to the **Map** tab and press **Auto place**.

The app will try to find a compact and stable layout automatically.

After the first quick calculation, it may continue searching in the background and replace the layout if it finds a better one.

### 4. Check the result

After calculation, check:

* **Result** — general layout summary
* **Warnings** — possible problems
* **Auto-place log** — what was changed during auto-placement
* **Generators / Sources / Heat Sinks** — detailed building information

### 5. Save or share

Open **Saves** to save the layout locally on your device.

You can also export/import JSON if you want to share a layout or move it to another device.

### Notes

The app works fully offline.

All data is stored locally on your device.

This is an early version, so some UI parts may still be rough or partially untranslated.
